# Task Manager Web Application

A secure web-based task management application that allows users to manage personal tasks through a simple web interface.

This project is being built as part of my journey to strengthen my backend development skills using Java and the Spring ecosystem.

## Tech Stack

Frontend
- HTML5
- CSS3
- JavaScript

Backend
- Java
- Spring Boot
- Spring MVC
- Spring Security
- Spring Data JPA

Security
- BCrypt password hashing

## Current Features

- User registration
- User login
- Password change functionality
- Secure password storage using BCrypt

## Features In Development

- JWT-based authentication
- Task creation
- Task editing
- Task deletion
- User task dashboard

## Security

User passwords are securely hashed using the BCrypt algorithm before being stored in the database.

Future versions will incorporate JWT authentication to support stateless API security.

## Project Status

The project is currently in early development. Additional features and improvements will be added as development progresses.
